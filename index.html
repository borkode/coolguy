cool guy
Cross-server Discord chat bot
Get to know the other users of Discord

Cool guy is a Discord.js-powered bot user that helps users interact with other servers.
It does this by storing the server IDs in arrays, and then transferring message data
back and forth, throughout both servers.

?talk - Enlists your server in the queue to talk to another random server
?disconnect - Removes your server from the connection (if any)
?unqueue - Unenlists you from the queue, if your server is on there.
?bugreport {bug} - Find a bug? Submit it! We can't find all the bugs ourselves!

By not so cool guy#3258
Hosted on a Raspberry Pi 3
